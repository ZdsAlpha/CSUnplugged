Error Detection and Correction

Introduction

When data is stored on a disk or transmitted from one computer to another, we usually assume that it doesn’t get changed in the process. But sometimes things go wrong and the data is changed accidentally. This creates a problem and we wants to detect an error and correct it .This Activity is basically designed to teach error detection and correction in a computer without using the computer (Cs Unplugged) to primary school children. This will explain how to teach kids an error detection through drawing and using charts, cards, markers etc.
When someone write some text through keyboard in a computer and input code is generated which is in zeros and ones. When an input code sequence is changed accidently and sequence of zeros and ones changed, through error detection we usually find the one bit in code that causes a problem and fix it through error correction. This activity covers the instructions and demonstration to teach error detection to kids using physical things.

Instructions

Students and Instructor should follow the following instructions to correctly perform this activity. The instructions are also in the demonstration video also. • To perform this activity you need board maker, cards of two colors which are black and white.
• Draw three circles which are intersecting with each other.
• The black card represents 1 and the white card represents 0.
• The 4 cards are to be placed in the correct positions of supposed input sequence.
• If there are even number of black cards in a circle place white card to make black cards even.
• If there are odd number of black cards in a circle place black card to make black cards even.
• For demonstrating error detection replace one with card with opposite color and check the sequence.
• For correction add 3 more cards in the in remaining space of circles to check which card causes the error and correcting the sequence will tell about the particular card that causes the error.

Skills Required

The person should be able to understand the basic number counting, shapes and English alphabets. The person should also be able to read and write. The primary level students must have the basic knowledge of the following terms.
1) Computer
2) CPU(Central processing unit)
3) Computer Instructions
4) Sequence of numbers
5) Geometrical shapes
6) Display of the Computer
7) Components of computer ( keyboard ,mouse etc.)
